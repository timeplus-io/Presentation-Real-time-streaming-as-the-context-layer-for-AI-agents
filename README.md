# Real-time Streaming as the Context Layer for AI Agents

Interactive presentation exploring how streaming data infrastructure transforms AI agents from static response systems into proactive, context-aware production assistants.

## Overview
This project is a 15-slide interactive slide deck built with React and Framer Motion. It covers:
- The "Context Gap" in modern AI agents.
- Limitations of pull-based RAG architectures.
- The evolution towards **Context Engineering**.
- Deep dives into streaming architectures (Confluent, Timeplus, Redpanda).
- 6 Industry-specific use cases (Security, DevOps, Finance, IoT, CX, Supply Chain).

## Features
- **Modern UI**: Clean, black and Timeplus pink aesthetic.
- **Dynamic Animations**: Pulse effects, animated data packets, and slide transitions.
- **Interactive Infographics**: Visual representations of streaming pipelines.
- **Responsive Layout**: Designed to fit standard presentation viewports.

## Tech Stack
- **Framework**: React 19
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion (`motion/react`)
- **Icons**: Lucide React

## Getting Started

### Installation
```bash
npm install
```

### Running Locally
```bash
npm run dev
```
The presentation will be available at `http://localhost:3000`.
